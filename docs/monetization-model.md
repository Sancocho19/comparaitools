# Monetization model

## Highest value page types
1. pricing
2. versus
3. alternatives
4. best-for-role roundups
5. workflow guides

## Revenue layers
1. affiliate clicks from reviews, pricing, and alternatives
2. sponsor placements for premium visibility
3. newsletter sponsorship once traffic exists
4. B2B lead generation for enterprise tools

## CTA strategy
- top CTA: compare tools
- mid CTA: view pricing / visit tool
- bottom CTA: join newsletter or compare related tools

## Avoid
- too many display ads early
- intrusive popups that crush UX
- sponsor placement that makes editorial judgment look fake
